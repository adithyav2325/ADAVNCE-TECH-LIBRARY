**ADVANCE TECH LIBRARY SYSTEM**



**1)PROBLEM STATEMENT**

The traditional library systems face the issues of manual tracking and checking the availability of the books immediately and very slow process of borrowing entertaining the book. There is a need for the digital system which automates everything and lets the user to do all those stuffs at one place in a real time console environment



**2)SCOPE OF PROJECT**

The project as console based library management system developed using python language. It's demerits real world borrowing and returning the book environment the user can also manage his or her wallet by adding extra amount or looking at the deduction of amount when book is borrowed. The user can also track the due dates and extend the due dates by paying extra amount from his or her wallet. The scope of usage is limited to a single user using the concept of object oriented programming data structures during the runtime



**3)TARGET USER**

Students and the self learners are the main target of the system and also the library administrators who need technology the manage all their inventories and other stuff under a single system itself



**4)FEATURE**

* Browse library  - user can view the details of the book the amount and the duration of time the book can be borrowed 



* Wallet management -  the user can add extra amount or handle the amount deducted from his or her wallet after borrowing a book 



* Borrowing system -  the user can borrow book from the library and the amount will be automatically deducted from his or her wallet 



* Returning system -  the user can automatically return the book back to the library 



* Extension system -  the user can extend the due date by paying extra amount from his wallet for the borrowed book
