**ADAVNCE TECH LIBRARY**



**OVERVIEW**

The advanced tech library is a python based command line application designed for the digital library purpose. It uses object oriented programming principles and other basic essentials of python programming to manage the book inventory wallet of the user borrowing and returning transactions also. User can directly borrow the tech related books and manage there wallet in real time from this application



**FEATURES**

* Dynamic inventory -  it updates book availability on real time 
* Financial system - it has an integrated wallet management for borrowing and extending the due date to pay the amount 
* Transaction tracking -  the user can use this to track the books borrowed and the books he returned 
* Extension logics - there is an option for the user to extent his due date by paying extra amount 
* Validation of the input - error handling has also been taken care for the efficiency of the application.



**TECHNOLOGIES USED**

* Language - python 3.13.7
* Concepts  - classes, objects, lists, dictionary, exception handling, functions, methods, operators, conditional statements, looping statements 
* Libraries  - time.



**STEPS TO RUN PROGRAM**

1. Ensure python 3.13.7 is installed on the system your running the program 
2. Download the so and so.py file to your system 
3. Open a terminal or a comment prompt from your system 
4. Navigate to the location of the file 
5. Run the given comment python so and so.py 



**TESTING OF PROGRAM** 

* Select option 1 -  to browse the available books please note the ID of the book from the browsing details 
* Select option 2 - for login into your cart or bag for the details of borrowed books and the due date to return 
* Select option 3 -  to add extra amount to your wallet 
* Select option 4 - to exit from the menu
